1)Що таке Git?
Git - це система контролю над версіями

2)В якому статусі можуть перебувати файли при роботі з Git?
tracked / untracked

3) Для чого використовується команда add?
Для додавання файлів

4)Для чого використовується команда status?
Для перевірки статусу файлів

5)Для чого використовується команда commit?
Щоб зберегти версії файлів в данний момент часу

6)Для чого використовується команда init?
Для ініціалізації репозиторію

7)Що потрібно писати в коментарях комітів?
Всі зміни з минулих версій

8)Як переглянути список комітів?
git log

9)Для чого використовується команда checkout?
для навігації між гілками в репозиторії
